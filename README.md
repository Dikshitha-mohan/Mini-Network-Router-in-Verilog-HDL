# Mini-Network-Router-in-Verilog-HDL

Designed and simulated a packet-based Mini Network Router capable of forwarding incoming packets to multiple output ports based on destination address decoding. The project demonstrates key RTL design concepts including packet routing, combinational and sequential logic, state-based packet handling, and functional verification using a dedicated testbench.

Features:
• Packet destination decoding
• Multi-port packet forwarding
• Synchronous RTL design
• Functional simulation and waveform analysis
• Verilog testbench for verification

Tools Used:
• Verilog HDL
• EDA Playground
• Icarus Verilog
• EPWave / GTKWave

Skills:
RTL Design, Digital Design, Verilog HDL, Verification, FSM Design, VLSI Front-End Design
